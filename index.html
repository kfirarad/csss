How hard can it be to create a CSS pie chart?

width:0; height:0; border: 150px solid yellow; border-radius:50%; border-right-color: yellowgreen; transform: rotate(-45deg)

How about pseudo elements?

before: bg, padding, content, position, bottom50, left50
transform: skewY, scaleX, rotate, transform-origin

Some transform clarification

  • Scale
  • Rotate
  • Skew

It this a good solution?

A good solution holds 3 things:

  • Flexibility
  • Extensibility
  • Maintainability

גמישות, ניתן להרחבה, קל לתחזוקה

Flexibility

  • Possible to change?
  • Possible to change inline?
  • How many edits?

Extensibility

  • can I have multiple segments?
  • Can I animate it?
  • Can I apply effects (gradients, hover) ?

Maintainability

  • How much code?
  • Extra elements?
  • Straightforward
